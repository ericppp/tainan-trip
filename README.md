2025年12月12日至14日台南三天兩夜旅行行程規劃

## 🚀 線上瀏覽

👉 [點我開啟行程頁面](https://ericppp.github.io/tainan-trip/)

- ✅ 完整響應式設計 (RWD)

## 🌦️ 天氣整合

使用 [Open-Meteo API](https://open-meteo.com/) 提供台南地區天氣資訊：
- **即時溫度顯示**：Header 區域顯示當前台南天氣
- **三天天氣預報**：底部導航顯示各日天氣圖示
- **氣溫範圍**：顯示每日最低溫~最高溫（例：18°~25°）
- **WMO 天氣代碼**：符合國際氣象組織標準
  - 0 = 晴空 ☀️
  - 1 = 大致晴朗 🌤️
  - 2 = 部分多雲 ⛅
  - 3 = 陰天 ☁️
- **查詢日期**：精準查詢 2025-12-12 至 2025-12-14 三天預報

## 📝 行前準備清單

點擊 Header 右側的「📋 行前準備」按鈕，可開啟準備事項清單視窗：
- 自動讀取 `行前準備秘笈Check List.xlsx` 檔案
- 依 Excel 欄位（衣、食、住、行、育、樂等類別）分類顯示
- 使用 XLSX.js 庫進行前端解析
- 無需額外安裝，純網頁即可執行
