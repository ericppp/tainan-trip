# 果果台南小旅 🌸

2025年12月12日至14日台南三天兩夜旅行行程規劃

## 🚀 線上瀏覽

👉 [點我開啟行程頁面](https://yourusername.github.io/tainan-trip/)

## 📱 功能特色

- ✅ 完整響應式設計 (RWD)
- ✅ 莫蘭迪暖色系配色
- ✅ 台南即時天氣顯示
- ✅ 三天天氣預報圖示
- ✅ 互動式日程切換
- ✅ 一鍵 Google Maps 導航
- ✅ 手機優化體驗

## 📋 行程概要

### Day 1 (12/12 五) 🌤️
林口 → 溪湖糖廠 → 台南水交社 → 和逸飯店

### Day 2 (12/13 六) ☀️  
雛菊餐桌 → 市區散策 → 安平小旅行 → 牛五蔵

### Day 3 (12/14 日) 🌥️
台南早餐 → 台中中央公園 → 返回林口

## 🌦️ 天氣 API

使用 [Open-Meteo API](https://open-meteo.com/) 提供即時天氣資訊
- 即時溫度顯示
- 三天天氣預報
- 自動每 10 分鐘更新
